# ✨ UI/UX Design & Principles

## 🎨 Design Theme

The interface follows a modern, clean, and accessible aesthetic:

- **Typography:** **Inter** (Modern sans-serif) for all headers and body text.
- **Palette:** Periwinkle/Lavender hero gradient (`--hero-gradient`), high-contrast slate text, and white surfaces.
- **Micro-interactions:** Hover-responsive cards, translucent sticky navbar, and subtle transitions.

## 🏠 Landing Page

- **Hero:** Punchy call-to-action ("Access Macedonian Public Data through AI").
- **Example Queries:** Clickable cards for real-world scenarios (Inflation stats,人口 statistics).
- **Technology Showcase:** Interactive section on MCP and supported AI models (Claude, GPT, Gemini).
- **Responsive Layout:** Full mobile stack support with hidden/collapsible sidebars on small screens.

## 🧪 Demo Interface

- **Active Navigation:** Narrow, clean white sidebar on the left for query input.
- **Controls:** Dropdowns for AI model and Data Source selection.
- **Result Panels:** Tabbed view for answers:
    - **Natural Language:** AI-generated response.
    - **Structured Table:** Raw data transformed into an easy-to-read grid.
    - **JSON:** Developer-friendly raw data output.
- **Transparency:** "MCP Reasoning" toggle to show how the tool was chosen and utilized.
